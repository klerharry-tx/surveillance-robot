# Arduino Surveillance Robot

## Overview

A surveillance robot built using Arduino Uno, ultrasonic sensor, servo motors, and DC gear motors. The robot is designed to move in different directions and scan its surroundings using an ultrasonic sensor mounted on a servo motor.

## Hardware Components

- Arduino Uno
- Ultrasonic Sensor (HC-SR04)
- 2 Servo Motors
- 4 DC Gear Motors
- L298N Motor Driver
- Laser/Turret Module
- Push Buttons
- PCB
- Jumper Wires
- 4 × 18650 Batteries
- Toy Car Wheels
- Sunmica Board

## Features

- Obstacle Detection
- Servo-Based Scanning
- Four-Wheel Drive System
- Laser Module Integration
- Battery Powered
